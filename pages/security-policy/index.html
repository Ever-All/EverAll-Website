Security Policy

Last Updated: May 11, 2025

1. Security Overview

EverAll employs specific security measures to protect user accounts and maintain platform integrity. Our security approach focuses on preventing unauthorized access while collecting minimal necessary data.

2. Account Security Measures

We implement the following security features:

  • Device fingerprinting to identify trusted devices
  • Login location monitoring for suspicious activity detection
  • Strong password requirements
  • Session management and automatic timeouts

3. Data Collection for Security

We collect and store only essential security-related information:

  • Device fingerprints for recognizing your trusted devices
  • Login location data to detect unusual access patterns
  • Basic account credentials (email and password hash)
  • Session data for active login management

4. Payment Processing

All payment processing is handled through PayPal's secure infrastructure. EverAll does not store or process any payment information directly.

5. Security Notifications

Users will receive alerts for:

  • Logins from new devices or locations
  • Suspicious login attempts
  • Account credential changes
  • Security-related account updates

6. User Security Responsibilities

To maintain account security, users should:

  • Use strong, unique passwords
  • Monitor email for security notifications
  • Report suspicious account activities
  • Keep login credentials confidential

7. Security Updates

We regularly review and update our security measures. Users will be notified of significant changes to security features or data collection practices.
