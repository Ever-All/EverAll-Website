Accessibility Statement

Last Updated: May 17, 2025

EverAll is committed to ensuring digital accessibility for people with disabilities. We continually work to improve the user experience for everyone and apply the relevant accessibility standards.

1. Our Commitment

We strive to ensure that our platform meets WCAG 2.1 AA standards. This includes:

  • Clear, consistent navigation across all pages
  • Proper heading structure and semantic HTML
  • Sufficient color contrast for text and interactive elements
  • Alternative text for images and meaningful link descriptions
  • Keyboard navigation support

2. Key Features

2.1 Visual Accessibility

  • High contrast theme options
  • Resizable text without loss of functionality
  • Clear visual focus indicators
  • No content flashing more than three times per second

2.2 Keyboard Navigation

  • All features accessible via keyboard
  • Logical tab order
  • Visible focus indicators
  • Skip navigation links

3. Assistive Technology

Our platform is designed to be compatible with common assistive technologies, including:

  • Screen readers (NVDA, JAWS, VoiceOver)
  • Voice recognition software
  • Screen magnification tools
  • Various browser accessibility extensions

4. Known Limitations

While we strive for comprehensive accessibility, some areas may need improvement:

  • Some third-party integrations may have limited accessibility features
  • Certain real-time trading features may require alternative methods of access
  • Complex data visualizations offer text alternatives but may have limited interactivity

5. Feedback and Support

We welcome your feedback on the accessibility of EverAll. Please let us know if you:

  • Encounter any accessibility barriers
  • Need information in an alternative format
  • Have suggestions for improvement
