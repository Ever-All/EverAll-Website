Commands

Here you can find a list of all the commands available in EverAll.

  • /alert - Get the latest trading alerts.
  • /follow [trader] - Follow a specific trader.
  • /unfollow [trader] - Unfollow a specific trader.
  • /switch [trader] - Switch to a different trader.
  • /help - Get help with using EverAll.
